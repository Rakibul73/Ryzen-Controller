# Ryzen controller

## For users

### Pre-requisite

- Go to https://github.com/FlyGoat/RyzenAdj/releases and download the latest compiled version.
- Execute Ryzen Controller, go select ryzenadj.exe into settings tab.
- Enjoy!

## For developpers

### Pre-requisite

- NodeJS v10.15.1 or newer.

### Installation

```bash
$> cd project
$> npm install
$> npm start
```

### Build

```bash
$> cd project
$> npm run-script build
```
